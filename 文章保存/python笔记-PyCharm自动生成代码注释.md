

在pycharm里面创建.py文件时，就会自动生成模板信息，需要在下面进行设置的

![pycharm里面的设置](https://www.github.com/Valuebai/my-markdown-img/raw/master/小书匠/1561703165594.png)

``` python
#!/usr/bin/env python
# -*- coding: UTF-8 -*-
'''=================================================
@IDE    ：${PRODUCT_NAME}
@Author ：LuckyHuibo
@Date   ：${DATE} ${TIME}
@Desc   ：
=================================================='''



if __name__ == "__main__":
    pass
```

至于python中要使用
#!/usr/bin/python
还是使用
#!/usr/bin/env python
